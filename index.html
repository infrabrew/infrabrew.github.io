Built for performance.
Designed for developers.

SwiftLLM combines continuous batching, PagedAttention for optimal memory management, and optimized CUDA kernels to deliver maximum throughput with minimum latency.

Whether you need an OpenAI-compatible API server, batch processing, interactive chat, or fine-tuning with LoRA/QLoRA, SwiftLLM provides a unified, production-ready solution.

Air-Gapped Deployment

Bundle and deploy on networks with no internet access

🚀

Speculative Decoding

Draft model acceleration for faster token generation

from swiftllm import LLM, SamplingParams

# Load model with automatic GPU detection
llm = LLM("meta-llama/Llama-3-8B")

# Configure sampling
params = SamplingParams(
    temperature=0.7,
    top_p=0.9,
    max_tokens=256
)

# Generate
outputs = llm.generate(
    ["Explain quantum computing"],
    params
)

for output in outputs:
    print(output.text)

Get Started

Up and Running in Minutes

Three simple steps to start running LLMs locally.

1

Clone & Install

Clone the repo and run the installer. It handles Rust, Python, and CUDA setup automatically.

git clone https://github.com/infrabrew/swiftllm && cd swiftllm && bash install.sh
2

Download a Model

Use the CLI to download any supported model from HuggingFace Hub.

swiftllm download meta-llama/Llama-3-8B
3

Start Serving

Launch an OpenAI-compatible API server with a single command.

swiftllm serve --model meta-llama/Llama-3-8B --port 8000
