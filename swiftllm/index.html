Built for performance.
Designed for developers.

SwiftLLM combines continuous batching, PagedAttention for optimal memory management, and optimized CUDA kernels to deliver maximum throughput with minimum latency.

Whether you need an OpenAI-style API server, batch processing, interactive chat, or fine-tuning with LoRA/QLoRA, SwiftLLM provides a unified, production-ready solution.

Rust + Python Hybrid

Rust core via PyO3 + maturin abi3 wheels for Python 3.8–3.12

CPU, CUDA, ARM

CPU is default — CUDA opt-in with --gpu. Ships x86_64 and aarch64 wheels

Air-Gapped Deployment

Bundle wheels, source, and models into a signed tarball for offline installs

quickstart.py 
from swiftllm import LLM, SamplingParams

# Load model with automatic GPU detection
llm = LLM("meta-llama/Llama-3-8B")

# Configure sampling
params = SamplingParams(
    temperature=0.7,
    top_p=0.9,
    max_tokens=256,
)

# Generate
outputs = llm.generate(
    ["Explain quantum computing"],
    params,
)

for output in outputs:
    print(output.text)
Capabilities

Powerful features, built-in.

Everything you need to deploy and manage LLMs in production environments.

High Throughput

Continuous batching and efficient scheduling maximize tokens per second.

PagedAttention

Block-allocator KV cache with copy-on-write delivers optimal VRAM utilization.

Speculative Decoding

Draft-model acceleration and O(n) quickselect sampling for sub-second latency.

Multi-GPU Scaling

Tensor parallelism and pipeline parallelism scale seamlessly across NVIDIA GPUs.

OpenAI-Style API

API server with key auth, input validation, CORS, TLS, and Prometheus metrics. Compatible with most common OpenAI SDK calls — review the docs for supported endpoints.

Training & Fine-Tuning

LoRA, QLoRA, full fine-tuning with Muon, AdamW, and SGD optimizers.

Multi-Format Models

HuggingFace, GGUF, SafeTensors, and PyTorch formats with auto-detection.

Air-Gapped Install

Portable bundles with SHA256 verification for fully offline deployment.

CPU & ARM Ready

Ships portable wheels for x86_64, aarch64, Apple Silicon, and Graviton — CUDA is opt-in.

Portability

Runs everywhere.

Portable abi3 wheels for Python 3.8–3.12. CPU is the default; CUDA is opt-in.

Platform x86_64 aarch64 / arm64
Linux manylinux2014_x86_64 manylinux2014_aarch64
macOS macosx_10_15_x86_64 macosx_11_0_arm64 (M1+)

Build on your dev laptop, ship to Graviton, Raspberry Pi, Jetson, or Apple Silicon. CUDA 11.8+ unlocks GPU inference; ./install.sh --gpu enables it.

Get Started

Up and running in minutes.

Four simple steps to install, download a model, and start serving.

1

Clone & install

Installer auto-detects GPU, builds with Rust + maturin, installs [serve] extras.

$./install.sh
2

Download a model

Pull any supported model from HuggingFace Hub, including GGUF quantized variants.

$swiftllm download -m Qwen/Qwen2.5-7B
3

Run inference

Batch generate, chat interactively, or pipe prompts from a file.

$swiftllm generate -p "Say hi"
4

Serve an API

OpenAI-style server with key auth, CORS, and /metrics endpoint.

$swiftllm serve --port 8000
